## P0-1. [C-2 / G-19] claude Edge Function `--no-verify-jwt` + 1.3.0 일일 cap 제거 = 비용 폭주
- 위치: `supabase/functions/claude/index.ts:2`, `supabase/functions/claude/README.md:18`

- 수정 복잡도: 중간 (verify_jwt true + DB 기반 user-level 쿼터)
